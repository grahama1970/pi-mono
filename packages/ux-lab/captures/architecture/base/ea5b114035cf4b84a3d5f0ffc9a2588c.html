# Architecture Editor — Stitch Design Spec Visual architecture diagram editor where a human and AI agent co-edit system pipeline diagrams. ## Key Features - Visual source of truth for system architecture. - Version history (v1, v2, v3) with attribution (Human vs Agent). - Freeform drawing (boxes, arrows, text). - Details panel for component metadata (tech, latency, files, edges). ## Aesthetics - Excalidraw-meets-Figma. - Dark tactical HUD. - Background: #141414 - Surface: #1a1a1a - Text: #e2e8f0 - Accent: #7c3aed (purple) - Success: #00ff88 (green) - Fonts: Space Grotesk (headlines), JetBrains Mono (code/details) - 0px border radius (sharp corners).
